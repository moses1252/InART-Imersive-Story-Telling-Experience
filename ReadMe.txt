5/12/2022 How to operate our game
Once launching the game on the Oculus quest 2, you will start off in the menu:

To interact with the menu, point hand at the button you want to press, and press the trigger button on Oculus Controller.

Control setup for Farm and Kichen level:
Left joystick on left controller will move the player.
The grab buttons on the Oculus Controller will grab interactable items. 

To active wrist menu in level: press X
To active ray interacter while wrist menu is out: press A
To press Menu button, press the trigger button while Holding X and A
